# DieFits — Industrial Fitting Inventory

Cloud-synced fitting inventory tracker for tool & die shops.

## Features
- 🔐 Email/password login (Supabase auth)
- ☁️ Real-time cloud sync across all devices
- 📷 AI-powered photo scanning (Claude vision + web search)
- 🧠 AI learns from user corrections
- ⚠️ Low-stock alerts with email notifications
- 📦 Full inventory management (add/edit/delete/search)

## Setup
1. `npm install`
2. `npm run dev` (local development)
3. `npm run build` (production build)

## Deployment
Deployed via Vercel. Pushes to `main` auto-deploy.
